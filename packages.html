Pertanyaan Umum (FAQ)

Berapa minimal order untuk paket bundling?

Minimal order standar adalah 50 paket. Namun untuk paket VIP/Exclusive Hardbox, kami menerima pesanan mulai dari 20 paket dengan penyesuaian harga.

Apakah bisa request isi paket di luar katalog?

Tentu saja! Katalog ini hanya referensi. Anda bisa Mix & Match item apapun yang ada di halaman Produk kami. Silakan konsultasikan budget Anda ke Admin.

Berapa lama pengerjaan gift set custom box?

Estimasi pengerjaan lengkap dengan Box Custom adalah 7-14 hari kerja tergantung antrian dan kerumitan desain. Untuk kebutuhan urgent, silakan cek kategori Express.
